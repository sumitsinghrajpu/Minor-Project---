# 🎓 Student Marks Prediction System

This is an AI/ML-based web application built using **Streamlit** that predicts student performance and suggests weak/strong subjects based on academic data.

---

## 🚀 Features

- 🔢 Manual student data entry
- 📂 CSV file upload for bulk prediction
- 🤖 Machine Learning based marks prediction
- 📊 Subject-wise performance analysis
- 📈 Data visualization using Matplotlib
- 💡 Strong & weak subject recommendation

---

## 🧠 Machine Learning Model

- Algorithm: Random Forest Regressor
- Input Features:
  - Mid Sem 1 Marks
  - Mid Sem 2 Marks
  - SGPA
  - CGPA
  - Study Hours
  - OS, CD, ML, SD Marks

- Output:
  - Final Semester Marks
